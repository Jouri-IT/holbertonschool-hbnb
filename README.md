# holbertonschool-hbnb
Developing a full-stack accommodation rental platform inspired by Airbnb.

## Project Overview

This project is a simplified clone of the Airbnb platform developed as part of the Holberton School curriculum. The first phase focuses on building the application's core using Python, object-oriented programming, JSON serialization, and a command-line interpreter.

## Features

- Command-line interface
- Object creation and management
- JSON serialization and deserialization
- File storage
- Object-oriented design

## Technologies

- Python 3
- OOP
- JSON
- Git & GitHub
- Linux
